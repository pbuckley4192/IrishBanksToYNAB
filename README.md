# Irish Bank Statements To YNAB Converter

A Flask app that you can deploy and use to convert Bank Statements to YNAB / Homebank format.

## Banks Supported
* Bank of Ireland
* AIB

**Note:** Ulster Bank is supported by YNAB already.
